Import Data

This page allows you to import projects, tasks, and work in a variety of formats.

Import CSV Files
  • Import Projects from CSV

    • All CSV data must be well-formed and contain column headers. At least one column must be defined in the csv file called "name" with the name of each project. Additional fields can be "description", "phase", and "status". Other fields will be ignored.

  • Import Work from CSV

    • All CSV data must be well-formed and must contain column headers. A minimum of two columns are required, one called 'name' which contains the name of the project a worklog is associated with, and another called 'description' with a description of the work. Optional fields are 'datestamp' and 'duration' (in hours). Any extra fields will be ignored.

Import a List of Projects
  • Import Projects

    • This option allows you to import a simple list of projects. The first line in your list of projects should say "name" and each project name should be on a new line. If a project name has spaces, it should be in quotes. Example:

    name
"My First Project"
"My Second Project"
